Weather App

A simple and user-friendly weather application that provides current weather information based on your location or any city you search for.

Features
- Get real-time weather updates
- Search weather by city name
- Displays temperature, humidity, wind speed, and weather conditions
- Responsive and clean user interface

Installation
1. Clone the repository:
   git clone https://github.com/yourusername/weather-app.git
2. Navigate to the project directory:
   cd weather-app
3. Open the index.html file in your browser to run the app.

Usage
- Enter a city name in the search box and click the search button or press Enter.
- The app fetches and displays the current weather details for the searched city.
- By default, it shows weather for Chennai.

Technologies Used
- HTML
- CSS
- Vanilla JavaScript
- OpenWeatherMap API

API Key Setup
To use the OpenWeatherMap API, you need to get your own API key from https://openweathermap.org/api.

1. Sign up and generate your API key.
2. Replace the `apiKey` value in the JavaScript code with your API key.
